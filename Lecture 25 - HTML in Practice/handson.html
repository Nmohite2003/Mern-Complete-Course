JavaScript MCQs (Multiple Choice Questions)

Test your JavaScript knowledge from easy to advanced questions.

1. Which of the following is a JavaScript data type?

a) Number

b) String

c) Boolean

d) All of the above

Answer: d

2. How do you write "Hello World" in an alert box?

a) msg("Hello World");

b) alertBox("Hello World");

c) alert("Hello World");

d) msgBox("Hello World");

Answer: c

3. How do you create a function in JavaScript?

a) function = myFunction()

b) function myFunction()

c) function:myFunction()

d) create function myFunction()

Answer: b

4. How do you call a function named "myFunction"?

a) call myFunction()

b) myFunction()

c) call function myFunction()

d) Call.myFunction()

Answer: b

5. How to write an IF statement in JavaScript?

a) if i = 5 then

b) if i == 5 then

c) if (i == 5)

d) if i = 5

Answer: c

6. How can you add a comment in JavaScript?

a) <!-- This is a comment -->

b) // This is a comment

c) ** This is a comment **

d) ## This is a comment ##

Answer: b

7. Which operator is used to assign a value to a variable?

a) -

b) =

c) *

d) ==

Answer: b

8. Which built-in method removes the last element from an array?

a) last()

b) get()

c) pop()

d) None of the above

Answer: c

9. How do you round the number 7.25 to the nearest integer?

a) round(7.25)

b) Math.round(7.25)

c) Math.rnd(7.25)

d) rnd(7.25)

Answer: b

10. What is the correct way to write a JavaScript array?

a) var colors = ["red", "green", "blue"]

b) var colors = (1:"red", 2:"green", 3:"blue")

c) var colors = "red", "green", "blue"

d) var colors = 1 = ("red"), 2 = ("green"), 3 = ("blue")

Answer: a

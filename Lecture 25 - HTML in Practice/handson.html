JavaScript MCQs (Multiple Choice Questions)

Test your JavaScript knowledge from easy to advanced questions.

  1. Which of the following is a JavaScript data type?

    1. Number
    2. String
    3. Boolean
    4. All of the above

    Answer: d

  2. How do you write "Hello World" in an alert box?

    1. msg("Hello World");
    2. alertBox("Hello World");
    3. alert("Hello World");
    4. msgBox("Hello World");

    Answer: c

  3. How do you create a function in JavaScript?

    1. function = myFunction()
    2. function myFunction()
    3. function:myFunction()
    4. create function myFunction()

    Answer: b

  4. How do you call a function named "myFunction"?

    1. call myFunction()
    2. myFunction()
    3. call function myFunction()
    4. Call.myFunction()

    Answer: b

  5. How to write an IF statement in JavaScript?

    1. if i = 5 then
    2. if i == 5 then
    3. if (i == 5)
    4. if i = 5

    Answer: c

  6. How can you add a comment in JavaScript?

    1. <!-- This is a comment -->
    2. // This is a comment
    3. ** This is a comment **
    4. ## This is a comment ##

    Answer: b

  7. Which operator is used to assign a value to a variable?

    1. -
    2. =
    3. *
    4. ==

    Answer: b

  8. Which built-in method removes the last element from an array?

    1. last()
    2. get()
    3. pop()
    4. None of the above

    Answer: c

  9. How do you round the number 7.25 to the nearest integer?

    1. round(7.25)
    2. Math.round(7.25)
    3. Math.rnd(7.25)
    4. rnd(7.25)

    Answer: b

  10. What is the correct way to write a JavaScript array?

    1. var colors = ["red", "green", "blue"]
    2. var colors = (1:"red", 2:"green", 3:"blue")
    3. var colors = "red", "green", "blue"
    4. var colors = 1 = ("red"), 2 = ("green"), 3 = ("blue")

    Answer: a
